## 常用的元字符
```
    .匹配除换行符以外的任意字符
    \w匹配字母或数字或下滑线或汉字
    \s匹配任意的空白符
    \d匹配数字
    \b匹配单词的开始或结束
    ^匹配字符串的开始
    $匹配字符串的结束

```


## 正则匹配规则
```
1  /a/ 匹配a
   /dog/.test('old dog')//true

2. 点字符（.）
   c.t 匹配c和t之间的包含任意一个字符的情况

3. 位置字符
  ^ 表示字符串的开始位置
  $ 表示字符串的结束位置

    从开始位置到结束位置只有test
   /^test$/.test('test test')//false

4. 选择符（|）  或关系
   / 11|22/.test('453211') //true

   可以匹配b或c，可以使用圆括号
   a和b之间有一个空格或者一个制表符
   /a( |\t)b/.test('a\tb') //true 

5. 转义符
   若要匹配+，就要写成\+
   /1\+1/.test('1+1') //true

   若使用RegExp方法生成正则对象，转义需要使用两个反斜杠，因为字符串内部会先转义一次
   (new RegExp('1\\+1')).test('1+1')

6. 特殊字符
    \cX 表示Ctrl-[X]，其中的X是A-Z之中任一个英文字母，用来匹配控制字符。
    [\b] 匹配退格键(U+0008)，不要与\b混淆。
    \n 匹配换行键。
    \r 匹配回车键。
    \t 匹配制表符 tab（U+0009）。
    \v 匹配垂直制表符（U+000B）。
    \f 匹配换页符（U+000C）。
    \0 匹配null字符（U+0000）。
    \xhh 匹配一个以两位十六进制数（\x00-\xFF）表示的字符。
    \uhhhh 匹配一个以四位十六进制数（\u0000-\uFFFF）表示的 Unicode 字符。

7. 字符类
  1）脱字符(^)
   [xyz]表示x、y、z之中任选一个匹配
   /[abc]/.test('apple')//true
   [^xyz]表示除了x、y、z之外都可以匹配
   /[^abc]/.test('hello world')//true 

   2)连字符(-)
   [abc]可写成[a-c], [0123456789]可写成[0-9]
   /[a-z]/.test('b') //true



 8. 预定义模式
    \d 匹配0-9之间的任一数字，相当于[0-9]
    \D 匹配所有0-9以外的字符，相当于[^0-9]
    \w 匹配任意的字母、数字和下划线，相当于[A-Za-z0-9_]
    \W 除所有字母、数字和下划线以外的字符，相当于[^A-Za-z0-9_]
    \s 匹配空格（包括换行符、制表符、空格符等），相等于[ \t\r\n\v\f]
    \S 匹配非空格的字符，相当于[^ \t\r\n\v\f]
    \b 匹配词的边界
    \B 匹配非词边界，即在词的内部

    /\s\w*/.exec('hello world')// [" world"]   \s表示空格，故匹配结果会包括空格
    var html = "<b>Hello</b>\n<i>world!</i>";
    /.*/.exec(html)[0]   //"<b>Hello</b>" 
    字符串html包含一个换行符，结果点字符（.）不匹配换行符 


    var html="<b>Hello</b>\n<i>world!</i>";
    /[\S\s]*/.exec(html)[0]  // "<b>Hello</b>\n<i>world!</i>"
     [\S\s]指代一切字符

9. 重复类
    {n}恰好重复n次
    {n,}至少重复n次
    {n,m}重复不少于n次，不多于m次
    /hel{2}o/.test('hello') //true

10. 量词符
    ? 问号表示某个模式出现0次或1次，等同于{0, 1}
    * 星号表示某个模式出现0次或多次，等同于{0,}
    + 加号表示某个模式出现1次或多次，等同于{1,}

    /h?ello/.test('hello') //true  0次或1次
    /h+ello/.test('hello') //true  1次或多次
    /h*ello/.test('hello') //true  0次或多次

    Windows\d+ 匹配Windows后面跟1个或更多数字

11. 贪婪模式 
    上述量词默认情况下为最大可能匹配，即匹配直到下一个字符不满足匹配规则为止

    var s='aaaa';
    s.match(/a+/)  //["aaaa"]

    若想改为非贪婪模式，可以在量词符后面加一个问号
     var s='aaaa';
    s.match(/a+?/)  //["a"]


    hi Lucy    \bhi\b.*Lucy\b
    .是另一个元字符，匹配除了换行符以外的任意字符
    *它是数量，它指定*前面的内容可以连续重复使用任意次以使整个表达式得到匹配
    .*连在一起就意味着任意数量的不包含换行的字符

    .*?就意味着匹配任意数量的重复，但是在能使整个匹配成功的前提下使用最少的重复

    c.*b 匹配最长的以c开始，以b结束的字符串  若搜索hdecdcd 会匹配整个字符串
    c.*?b 匹配最短的，以c开始，以b结束的字符串  hdecd cd
```
## 字符类
```
    \(?0\d{2}[)-]?\d{8} 可以匹配几种格式的电话号码
    如(010)12345678, 023-22334466, 01348596789
    首先是一个转义字符\(,它能出现0次或1次(?),然后是一个0，后面跟着2个数字(\d{2})，然后是)或-或空格中的一个，它出现1次或不出现(?)，最后是8个数字(\d{8})。

    \(0\d{2}\)[- ]?\d{8}|0\d{2}[- ]?\d{8}  可匹配(010)-12345678,(010)12345678 或010-12345678,01012345678
```
